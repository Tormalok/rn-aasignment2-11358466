## **Components Usage**

1. **Separator** : In the login page, the this component renders a line with a text in the middle separating the  
   input fields from the icons.

1. **SocialButtons** : This component renders three different icons as buttons in the login page.

1. **Header** : In the homescreen page, this component renders the name and email received from  
   the user as well as the profile image.

1. **SearchSection** : It displays both the search field and the filter button.

1. **CardList** : Responsible for rendering the whole section below the search bar and the filter button.

1. **JobCard** : This component contains the data for rendering the cards in the cardlist component.
